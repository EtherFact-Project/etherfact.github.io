Mining EtherFact

EtherFact uses the ethash proof of work algorithm, the same algorithm used in Ethereum and Ethereum.

Solo Mining

Follow the guide in Installation to either set up a Parity or Geth full node. After that, you can use any ethash miner (such as ethminer) to mine EtherFact. To mine using your GPU, run ethminer -G -F http://localhost:8545.

Pools

Pool mining helps miners to compensate “bad luck” and share “good luck” when it comes. Help is appreciated to keep the Dev Mining Pool running.
